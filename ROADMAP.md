# Roadmap

## CS Hub — Product Roadmap

---

## Phase 1 — Foundation MVP (Sprints 1–2)

**Goal:** Working application with core client management and mock communication data.

### Sprint 1
- User login via Microsoft Entra ID
- Dashboard — My Day view
- Client list and client profile
- Contacts management
- PostgreSQL database setup
- Vercel deployment pipeline

### Sprint 2
- Outlook email sync (mock data MVP)
- Email flagging and action tracking
- Attachment linking
- Email to client auto-association
- Teams meeting notes (mock data MVP)
- Follow-up task creation

---

## Phase 2 — Document Generation (Sprints 3–4)

**Goal:** Generate SOWs, POCs, and Quotes from templates populated with client data.

### Sprint 3
- Quote Builder
- SOW Builder
- POC Builder
- Proposal Builder
- PDF and Word export
- Document version control

### Sprint 4
- Contract storage
- Document template management
- Client branding on documents
- Document timeline linking

---

## Phase 3 — AI Requirement Analysis (Sprints 5–6)

**Goal:** AI reads communications and generates structured developer-ready requirements.

### Sprint 5
- Requirement Capture Engine
- Email and document parsing
- User story generation
- Acceptance criteria generation
- Developer task breakdown
- Missing information detection — Ask the Customer

### Sprint 6
- Ticket package builder
- Copy and export ticket output
- Ticket review and approval workflow
- End-to-end traceability linking

---

## Phase 4 — Relationship Intelligence (Sprints 7–8)

**Goal:** Customer health scoring, engagement tracking, and proactive follow-up recommendations.

### Sprint 7
- Customer Health Dashboard
- Health score engine
- Relationship timeline
- Smart reminders

### Sprint 8
- Customer pulse indicators
- Milestone detection and celebration prompts
- AI Relationship Coach — meeting preparation
- Proactive engagement recommendations

---

## Phase 5 — Project Intelligence (Sprints 9–10)

**Goal:** Organisational memory, recurring project detection, and asset reuse.

### Sprint 9
- Project Intelligence engine
- Similar project detection
- Reuse score estimation
- Asset surfacing — SOWs, quotes, URLs, test cases

### Sprint 10
- Customer Knowledge Library
- Customer Playbooks
- Before You Start assistant
- Annual campaign recognition
- Knowledge base search

---

## Phase 6 — Integrations (Sprint 11+)

- Real Microsoft Graph integration (Outlook, Teams, OneDrive, SharePoint)
- GitHub Issues direct push
- Azure DevOps integration
- Global search across all data sources
- Reporting and Analytics
- Administration module

---

## Future Enhancements

- Mobile application
- Customer-facing portal
- AI-generated release notes
- Automated customer satisfaction surveys
- Revenue tracking and forecasting
- Multi-tenancy for commercial productisation
