# User Stories

## CS Hub — User Stories by Epic

---

## Epic 1 — Authentication and User Management

- As a CSM, I want to log in using my Microsoft account so that I do not need a separate password.
- As an admin, I want to assign roles to users so that access is controlled appropriately.
- As a user, I want my session to persist so that I do not have to log in repeatedly during the day.
- As an admin, I want to deactivate a user without deleting their data so that history is preserved.

---

## Epic 2 — Client Management

- As a CSM, I want to create a client profile so that all client information is stored in one place.
- As a CSM, I want to add multiple contacts per client so that I know who to reach at each company.
- As a CSM, I want to store production, UAT, and staging URLs per client so that I can find them instantly.
- As a CSM, I want to see a client timeline so that I can review the full relationship history before a meeting.
- As a CSM, I want to flag a primary contact per client so that I always know who the main point of contact is.

---

## Epic 3 — Outlook Integration

- As a CSM, I want emails from a client to appear in their profile so that I do not have to search Outlook.
- As a CSM, I want to flag an email as requiring action so that it appears in my Work Inbox.
- As a CSM, I want to link an email to a project so that all project communication is traceable.
- As a CSM, I want attachments from emails to be stored against the client so that I can find them later.

---

## Epic 4 — Microsoft Teams Integration

- As a CSM, I want meeting notes from Teams to appear in the client timeline so that nothing is lost.
- As a CSM, I want AI-generated meeting summaries so that I do not have to write them manually.
- As a CSM, I want action items from meetings to become follow-up tasks automatically.
- As a CSM, I want to open a Teams chat with a client directly from their profile.

---

## Epic 5 — Document Management

- As a CSM, I want to store all documents against a client and project so that everything is in one place.
- As a CSM, I want version control on documents so that I can see what changed between versions.
- As a CSM, I want to export any document as PDF or Word so that I can send it to clients.
- As an admin, I want to manage document templates so that all CSMs use consistent formatting.

---

## Epic 6 — Quote Builder

- As a CSM, I want to create a quote with auto-populated client details so that I do not retype information.
- As a CSM, I want to add line items with quantities and rates so that the total calculates automatically.
- As a CSM, I want to set a quote validity date so that clients know when it expires.
- As a CSM, I want to reuse a previous quote as a starting point so that I save preparation time.

---

## Epic 7 — SOW Builder

- As a CSM, I want to generate an SOW from a template so that the structure is always consistent.
- As a CSM, I want client details and project scope to auto-populate so that I only add the specifics.
- As a CSM, I want to include assumptions and exclusions so that scope is clearly defined.
- As a CSM, I want the SOW to export as a professional Word document so that it is client-ready.

---

## Epic 8 — POC Builder

- As a CSM, I want to create a POC document with scope and success criteria so that expectations are set.
- As a CSM, I want to link the POC to a project so that it appears in the project timeline.
- As a CSM, I want to reuse a previous POC structure so that I start from a proven baseline.

---

## Epic 9 — AI Requirement Analysis

- As a CSM, I want to paste an email and have the AI extract requirements so that I do not do it manually.
- As a CSM, I want the AI to classify a request as feature, bug, enhancement, or support automatically.
- As a CSM, I want the AI to generate user stories from a customer email so that developers receive structured input.
- As a CSM, I want the AI to flag missing information and draft a clarification email so that tickets are complete.
- As a CSM, I want to review all AI-generated content before it is exported so that I remain in control.

---

## Epic 10 — Developer Ticket Builder

- As a CSM, I want to generate a complete ticket package from requirements so that developers have everything they need.
- As a CSM, I want to edit any field in the ticket package before exporting so that quality is maintained.
- As a CSM, I want to copy the ticket package to clipboard so that I can paste it into any system.
- As a CSM, I want to export the ticket package as a Word document so that it can be shared or filed.
- As a CSM, I want each ticket to reference the original customer email so that context is never lost.

---

## Epic 11 — Customer Health Dashboard

- As a delivery manager, I want to see all clients ranked by health score so that I know where to focus.
- As a CSM, I want to see why a client has a specific health status so that I understand the reasoning.
- As a CSM, I want AI-recommended next actions per client so that I know what to do without guessing.
- As a CSM, I want to be notified when a client enters At Risk status so that I can act immediately.

---

## Epic 12 — Relationship Intelligence

- As a CSM, I want the system to detect client milestones so that I can acknowledge them appropriately.
- As a CSM, I want AI-drafted messages for milestone events so that I can send them with one click.
- As a CSM, I want a meeting preparation summary so that I walk into every call fully briefed.
- As a CSM, I want smart reminders for clients I have not contacted recently so that no one is forgotten.

---

## Epic 13 — Project Intelligence

- As a CSM, I want the system to detect when a new request is similar to previous work so that I can reuse assets.
- As a CSM, I want to see a reuse score per asset so that I can decide what to carry forward.
- As a CSM, I want a Before You Start summary when opening a new project so that I begin with context.
- As a CSM, I want recurring annual projects to be recognised automatically so that preparation time is reduced.

---

## Epic 14 — Knowledge Management

- As a CSM, I want a knowledge library per client containing all reusable assets so that nothing has to be recreated.
- As a CSM, I want the system to build a client playbook over time so that onboarding a new CSM is fast.
- As a CSM, I want to store lessons learned against a project so that the same mistakes are not repeated.
- As a CSM, I want all assets searchable from one place so that I never have to browse folders.

---

## Epic 15 — Global Search

- As a CSM, I want to search across all clients, projects, emails, and documents from one search bar.
- As a CSM, I want to filter search results by type, client, and date so that I find things quickly.
- As a CSM, I want attachments to be included in search results so that files are never lost.

---

## Epic 16 — Reporting and Analytics

- As a delivery manager, I want a report of all active projects and their status so that I have full visibility.
- As a delivery manager, I want to see quote conversion rates so that I can measure sales effectiveness.
- As a delivery manager, I want customer engagement trends over time so that I can identify patterns.
- As an admin, I want to export any report as Excel or PDF so that it can be shared with leadership.

---

## Epic 17 — Administration

- As an admin, I want to manage users and roles so that access is correctly controlled.
- As an admin, I want to manage document templates so that all outputs are consistently branded.
- As an admin, I want to view an audit log of all system changes so that accountability is maintained.
- As an admin, I want to configure AI settings including model and prompt preferences so that outputs meet our standards.
